# Contributing

When contributing to this repository, please first discuss the change you wish to make via issue or discussions,
with the owners of this repository before making a change. 

Please note we have a [code of conduct](CODE_OF_CONDUCT.md), please follow it in all your interactions with the project.

## Pull Request Process

1. Fork this repository and create your feature branch (git switch -c my-new-feature).
2. Before submitting your patch, make sure to read https://github.com/poac-dev/poac?tab=readme-ov-file#contributing.
3. For the commit message style, follow https://git-scm.com/docs/SubmittingPatches#describe-changes.
